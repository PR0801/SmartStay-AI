# SmartStay AI Frontend

SmartStay AI is a student housing marketplace aligned with SDG 11. It helps students browse verified accommodations, shortlist stays, request bookings, and receive AI-ranked recommendations based on budget, location, room type, amenities, and verification score.

## Available Scripts

### `npm start`

Runs the frontend at `http://localhost:3000`.

### `npm run build`

Builds the app for production into the `build` folder.
